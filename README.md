 # Overview Of Markdown Note-Taking App
 Markdown Note-Taking App is a tool that allows users to write and organize notes using Markdown, a simple markup language that converts plain text into formatted text.

 # Objective
 This topic aim to provide a simple and structured way to take and manage notes using Markdown.
